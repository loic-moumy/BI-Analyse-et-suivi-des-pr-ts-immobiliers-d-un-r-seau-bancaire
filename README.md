## Dashboard BI : Analyse des prêts immobiliers

Contexte

Dans le cadre d’un projet de Business Intelligence, ce tableau de bord a été développé afin d’analyser l’activité de prêts immobiliers d’un réseau bancaire fictif (Crédit Breton).

L’objectif est de fournir aux responsables de la banque un outil décisionnel interactif permettant de suivre les prêts accordés et refusés selon plusieurs dimensions d’analyse.

Objectifs du projet

Suivre l’évolution des prêts immobiliers accordés et refusés

Analyser l’activité selon la dimension temporelle

Identifier les zones géographiques les plus actives

Mettre à disposition un dashboard interactif pour l’aide à la décision

Architecture des données

Le projet repose sur la mise en place d’un Data Warehouse modélisé en schéma en étoile.

Table de faits

Prêts immobiliers

Dimensions

Temps

Localisation

Type de prêt

Statut du prêt (accepté / refusé)

Cette modélisation permet d’optimiser l’analyse multidimensionnelle dans Power BI.

Étapes du projet
1. Préparation des données

Nettoyage et transformation des données

Structuration du modèle de données

2. Modélisation

Création d’un modèle en étoile

Mise en place des relations entre tables

3. Data Visualisation

Création de plusieurs visualisations interactives :

évolution du nombre de prêts dans le temps

répartition des prêts par localisation

comparaison prêts acceptés vs refusés

indicateurs clés (KPIs)

Dashboard

Le tableau de bord permet :

une exploration interactive des données

l’analyse par période et zone géographique

l’identification des tendances de l’activité de prêt

Technologies utilisées

Power BI

Modélisation dimensionnelle (schéma en étoile)

Data Visualisation

Résultat

Le projet aboutit à un dashboard décisionnel fonctionnel permettant aux responsables de la banque de suivre et analyser l’activité de prêts immobiliers.
